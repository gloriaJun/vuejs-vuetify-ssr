## Project Install
 ``` bash
npm install # or yarn
```

#### Build Setup for develop
``` bash
# serve in dev mode, with hot reload at localhost:8080
npm run dev
```

#### Build Setup for production
``` bash
# build for production
npm run build

# serve in production mode
npm start
```
